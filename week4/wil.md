:: 로지스틱 회귀 그리고 확률적 경사하강법 :: 
-로지스틱 회귀
로지스틱 회귀란 분류 모델로서 선형방정식을 학습한다. 
다중회귀의 선형방정식을 이용하며 
(z = w1*x1 + w2*x2 + w3*x3 + w4*x4 + w5*x5 + b)
여기서 z가 확률값이 되어야 함 -> 0~1 사이어야함
z를 0에서 1사이의 값으로 만들기 위해서는 음의무한대를 0으로 양의 무한대를 1로 설정해야 한다. 이 때 시그모이드 함수를 이용하면 편리하다. 샘플에 대한 예측확률을 모두 구하고 그 중 가장 높은 확률의 샘플로 테스트 데이터의 답을 예측한다.

-확률적 경사하강법(SGD)
경사하강법은 훈련세트 데이터를 가지고 경사가 없는 최저점을 찾아서 이동한다. 이 때 확률적 경사하강법은 전체 훈련데이터가 아닌 일부만 골라서 사용하기 때문에 확률적이라는 말이 붙는다.
전체 데이터셋에서 샘플을 하나씩 골라 경사를 따라 내려가고 전체 데이터셋을 다 쓰면 다시 전체 데이터셋으로 경사하강법을 진행한다. 이렇게 한 번 전체 데이터셋을 다 쓰는 과정을 에포크라고 한다. 
손실함수란 실제 데이터 값과 모델이 예측한 값의 차이를 나타내는 함수로 손실함수의 값이 작을수록 좋아 경사하강법이 이용된다. 

-과소/과대적합
과소적합은 에포크수가 너무 적어 최저점을 찾지 못한경우, 과대적합은 에포크 수가 너무 커 정확도가 올라가지 않는 경우이다. 따라서 에포크 수는 적절하게 조절해야한다.
